# Memory Cards

This is a simple memory card game made with React, Vite and Material-UI.

## Installation

1. Clone the repository to your local machine:

  Run in  bash
    git clone https://github.com/dizsel74/memory-cards.git

2. Get into the directory 
    cd memory-cards

3. Install all the dependencies
    yarn install

4. Star the server
    yarn dev

5. Open in browser
   Ctrl + Click over the URL that ahows in the terminal
